# bolt

## チェックリスト

## PC, TAB, SP 共通

- デザイン通りの見た目になっているか
- margin と padding の使い分けはできているか
  - margin:兄弟(隣接している)要素同士の余白
  - padding:要素の内側の余白
- コンテンツ幅の指定はできているか
- a タグのクリック可能領域は適切か
  - デベロッパーツールで要素がはみ出たり、小さくなったりしていないか確認

## PC

- 指定の meta 情報が head タグ内に定義されているか
- フォントの指定は適切か・デザインと同じか
- Giztech の「基本構造」に則した構造になっているか
- 背景色は画面を広げても幅いっぱいになるか
- それぞれのコンテンツでの上下の余白のとり方は適切か
- ホバーでの見た目の変化は指示通りできているか
  - ホバーしたとき、ホバーが外れたとき
- ホバーでの見た目の変化があるものは a タグで作っているか
  - href=""の記述も必要

## TAB, SP

- PC の見た目に影響を与えていないか
- メディアクエリをまたいで同じ要素に対して同じ記述をしていないか
- 画面幅を超える横幅の固定値での指定がないか
